## Building App

### Server

1. Build `go get`
2. Add `.env` files
3. Run `go run`
4. Check on `localhost:4000/ping`

### Web

1. Build - `yarn`
2. Run - `yarn dev`
3. Check on `localhost:5000`


Deps required.

- `redis`
